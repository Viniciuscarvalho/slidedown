# SlideDown
## Write Markdown. Get Beautiful Slides.

---

## The Problem

> 90% of presentations are forgettable. Most tools make it worse.

- Death by bullet points
- Ugly default themes
- Hours wasted on formatting
- No version control for slides

<!-- notes: Start with the pain point - everyone relates to bad presentations -->

---

## The Solution

**SlideDown** converts your markdown into beautiful, professional presentations — instantly.

Write content in markdown. Pick a theme. Export to Keynote or PowerPoint.

No design skills required.

---

## How It Works

| Step | Action | Time |
|------|--------|------|
| 1 | Write markdown | 5 min |
| 2 | Choose a theme | 10 sec |
| 3 | Run converter | 1 sec |
| 4 | Open in Keynote | instant |

---

## Themes Included

- **Midnight** — Dark navy, professional
- **Aurora** — Purple-teal, creative
- **Sunset** — Warm amber, energetic
- **Minimal** — Clean white, corporate
- **Forest** — Deep green, trustworthy
- **Brutalist** — Bold yellow, memorable

---

## Get Started

Just run one command:

```bash
node md2pptx.js talk.md talk.pptx midnight
```

That's it. Your presentation is ready.
